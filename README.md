# Sprint-9
Business Intelligence 2 (testing hypotheses)

- ## O que é o projeto?

Um projeto com BI, Teste A/B, estatística e gráficos .Compilação e análise de hipóteses. Testes A/B realizados para avaliar impactos. Resultados visualizados com gráficos e análise para priorizar implementações e foram testadas hipóteses estatísticas.

- ## Como instalar/usar

Basta acessar o arquivo .ipnyb com Google Collab ou VS Code

- ## Quais tecnologias foram usadas

Python básico
